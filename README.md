# 🌍 TerraRun

> **Run. Capture. Conquer.**

TerraRun is a location-based multiplayer fitness game that transforms every run into a real-world territory battle.

Instead of simply tracking your runs, TerraRun allows you to **capture**, **defend**, and **expand** your territory while competing against other runners in your city.

---

## 📱 Features

- 🗺️ Live Interactive Map
- 🏃 Real-time GPS Run Tracking
- 🌍 Territory Capture System
- ⚔️ Territory Battles
- 🏆 Global & Local Leaderboards
- 👤 User Profiles
- 🔥 Activity Feed
- 📈 Run Statistics
- 🔔 Real-time Notifications
- 🎮 Gamified Running Experience

---

## 🎯 Vision

Traditional running apps reward distance.

TerraRun rewards **strategy**.

Every street you run becomes your territory.

If another runner enters your territory, they can steal it.

Run more.

Capture more.

Become the ruler of your city.

# 🗺️ How It Works

```
User Starts Running
          │
          ▼
GPS Location Updates
          │
          ▼
Convert GPS → Territory
          │
          ▼
Capture Territory
          │
          ▼
Save to Firestore
          │
          ▼
Realtime Update
          │
          ▼
Map Refresh
```

# 💻 Getting Started

Clone the repository

```bash
git clone https://github.com/MusaleTejas/TerraRun.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npx expo start
```
